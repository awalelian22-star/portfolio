# 3D Models Directory

To display your own bike model:
1. Export your model from 3ds Max (or your modeling software) as a **.glb** file (GL Transmission Format Binary). **.max** files are not supported by web browsers.
2. Rename the exported file to `bike.glb`.
3. Upload it to this directory (`/public/models/`).

The application is currently configured to fall back to an abstract "Cyber Core" visual if `bike.glb` is not found or fails to load.
